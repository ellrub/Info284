## LSTM 
Complete script for training and evaluating a sentiment analysis model using a LSTM neural network.
### Libraries used
`numpy (Anaconda)` - `pip install numpy`

`pandas (Anaconda)` - `pip install pandas`

`tensorflow` - `pip install tensorflow`
* `tensorflow.keras.models`
    * `Sequential`
    * `load_model`
* `tensorflow.keras.layers`
    * `Dense`
    * `LSTM`
    * `Dropout`
    * `Embedding`
* `tensorflow.keras.preprocessing.text`
    * `Tokenizer`
* `tensorflow.keras.preprocessing.sequence`
    * `pad_sequences`

`sklearn.metrics (Anaconda)` - `pip install scikit-learn`
* `classification_report`
* `confusion_matrix`
